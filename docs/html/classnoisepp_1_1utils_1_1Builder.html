Detailed Description

A builder is a helper class that makes it easy to generate geometrical objects like a plane. You just have to specify the output size, destination and geometric specific options(like the geom size). Now you can set the source module and the build function will create the pipeline and element for you using the optimal system settings or create them manually and pass them to the build function.
